# Руководство мастера

Этот документ — для мастера салона. Чтение займёт 5 минут. Всё, что
видишь в боте, описано здесь. Если чего-то не нашёл — спроси
администратора.

## Как подключиться

1. Открой в Telegram бота [@userinfobot](https://t.me/userinfobot),
   напиши ему любое слово — он пришлёт тебе числовой Telegram ID
   (например, `123456789`).
2. Отправь этот ID администратору салона.
3. Администратор привяжет ID к твоей карточке мастера в боте.
4. Открой бота салона, нажми **/start** — ты увидишь приветствие
   «Кабинет мастера» с твоим именем и три кнопки внизу.

Если после `/start` видишь клиентский флоу (выбор категории «ручки /
ножки») — значит привязка ещё не сработала. Напиши администратору.

## Что ты видишь

Внизу экрана — постоянная клавиатура с тремя кнопками:

```
📋 Сегодня
📅 Мои записи
📆 Моё расписание
```

Все три открывают одно и то же «живое» сообщение-панель, которое
обновляется при каждом нажатии. Старые сообщения из разделов не
копятся — бот их сам прибирает.

## 📋 Сегодня

Список всех твоих записей на сегодня: ожидающие (🕐), выполненные (✅),
не пришедшие (🚫). Отменённые записи сюда не попадают.

Каждая запись — кликабельная. Жмёшь → открывается карточка:

```
📋 Запись #42
🕐 14:30 · 20 апреля, пн
👤 Айгуль Назарова
📞 +998 90 123-45-67
💅 маникюр с гель-лаком (1ч 30м)
📌 🕐 Ожидает
```

Действия в карточке (только пока запись `🕐 Ожидает`):

- **✅ Выполнено** — клиент пришёл, всё сделали.
- **🚫 Не пришёл** — клиент не явился. Запись остаётся, ты можешь
  потом перевести её в «Перенести» или «Отменить».
- **❌ Отменить** — запись аннулируется, клиент получает сообщение
  «мастер отменил запись» с датой, временем и услугой.
- **↔ Перенести** — открывается календарь на 7 дней вперёд, потом
  свободные слоты на выбранной дате. Клиент получает уведомление о
  переносе.

После смены статуса карточка перерисовывается с новым статусом;
финальные статусы (`✅ Выполнено`, `❌ Отменено`) действий больше не
предлагают, только «🔙 К записям».

Ошибся со статусом? Если запись ещё `🕐 Ожидает` — нажми нужную
кнопку. Если уже перевёл в `✅ Выполнено` или `❌ Отменено` — скажи
администратору, он вернёт обратно (у него в панели те же кнопки).

## 📅 Мои записи

Ближайшие ожидающие записи — сегодня и вперёд, до 30 штук,
сгруппированы по датам:

```
📅 Твои ближайшие записи

—— 20 апреля, пн ——
🕐 14:30 — Айгуль · маникюр с гель-лаком

—— 21 апреля, вт ——
🕐 10:00 — Мадина · педикюр
🕐 12:30 — Зухра · маникюр
```

Клик на запись → та же карточка, те же действия, что в «📋 Сегодня».

Если показаны все 30 и в нижней строке есть «… остальные позже» —
значит записей больше, они появятся в этом списке постепенно по мере
того, как ближние будут завершаться.

## 📆 Моё расписание

Две вещи на этом экране.

**Недельная сетка** — твои часы работы по дням недели:

```
📆 Твоё расписание

Понедельник   09:00 – 19:00
Вторник       10:00 – 18:00
Среда         — выходной
Четверг       09:00 – 19:00
Пятница       09:00 – 19:00
Суббота       10:00 – 16:00
Воскресенье   — выходной
```

**Часы меняет администратор.** Ты их не редактируешь — это
намеренно, чтобы не было рассинхрона с записями клиентов.

**Отгулы** — разовые выходные на конкретные даты. Это ты делаешь сам:

- **🌙 Поставить отгул** — открывается календарь на 14 дней вперёд,
  выбираешь день. Если на этот день нет твоих записей — отгул
  сохраняется, админы получают push-уведомление. Если записи есть —
  бот покажет сколько и попросит сначала попросить админа их
  перенести.
- **☀ Убрать отгул** — список будущих отгулов, выбираешь какой
  снять. Админы получают push-уведомление.

Будущие отгулы видны на этом же экране внизу:

```
Будущие отгулы:
· 25 апреля, пт
· 1 мая, чт
```

Повторно поставить отгул на ту же дату бот не даст — он скажет «на
эту дату уже стоит отгул».

## Что приходит от бота сам

- **Новая запись** — push-уведомление, как только клиент подтвердил
  бронь на тебя. Кнопка «✅ Принято» — убрать уведомление из чата.
- **Перенос администратором** — сообщение «админ перенёс запись X с
  старой даты/времени на новую». Запись в «📋 Сегодня» / «📅 Мои
  записи» обновится автоматически.
- **Отмена администратором** — push с записью, которую отменил
  админ.

Напоминания за 24 часа и за 2 часа приходят **клиентам**, не тебе.
У тебя есть «📋 Сегодня» — это и есть твоё напоминание.

## Типовые сценарии

**Клиент не пришёл к назначенному времени.**
Открой «📋 Сегодня» → клик на запись → **🚫 Не пришёл**. Если он
позвонил и попросил перенести — на той же карточке «↔ Перенести» и
выбери новую дату/время.

**Хочу выходной на следующий четверг.**
«📆 Моё расписание» → «🌙 Поставить отгул» → выбери дату. Если на
этот день есть записи — бот покажет сколько, и ты попросишь
администратора их перенести. После этого возвращаешься и ставишь
отгул повторно.

**Клиент позвонил, хочет перенести запись на завтра.**
«📅 Мои записи» → найди запись → клик → **↔ Перенести** → выбери
дату → выбери свободный слот. Клиент получит уведомление автоматически.

**Я по ошибке нажал «✅ Выполнено», а клиент ещё не пришёл.**
Эту запись сам уже не вернёшь в `🕐 Ожидает` — напиши
администратору, он это сделает через свою панель.

**Я не вижу новой записи, хотя клиент говорит что записался.**
Проверь «📋 Сегодня» — возможно запись на другой день, смотри «📅
Мои записи». Если всё равно не находишь — администратор проверит,
была ли запись вообще и на какого мастера. Бывает, что клиент
ошибочно выбрал другого мастера.
